移动端如何实现1px的细线

css

前言

在做移动端项目的时候，我们往往需要设置1px的边框，但是border-width:1px;效果出来的边框却往往比1px要粗。本文主要介绍移动端实现1px边框的几种实现方案。

实现方案：

1.使用小数来写

3.使用阴影渐变

使用box-shadow，设置白色阴影偏移实现

4.使用伪类 :before, :after 与 transform

构建1个伪元素, 将它的长宽放大到2倍, 边框宽度设置为1px, 再以transform缩放到50%.
优点: 其实不止是圆角, 其他的边框也可以这样做出来
缺点: 代码量也很大, 占据了伪元素, 容易引起冲突

5.border-image

缺点: 对于圆角样式, 将图片放大修改成圆角也能满足需求, 但这样无形中增加了border的宽度，存在多种边框颜色或者更改的时候麻烦。

这里只介绍这五种实现1px边框的方式，各种方式都有其优缺点，至于项目中具体使用何种方式，则由项目和开发者自己决定。
